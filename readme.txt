== Description ==

Load modular UIkit 3 front-end framework to your Wordpress. For more, visit the UIkit Documentation: https://getuikit.com/docs/installation

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/why-uikit-loader` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
